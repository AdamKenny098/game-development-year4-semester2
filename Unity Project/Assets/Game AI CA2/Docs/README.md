# AI Behaviour Tree Encounter

## Unity Version
Unity 6.3

## Packages Used
- Unity Behavior (Behaviour Tree system)

## Scene
- Scene Name: Game Scene  
- To run: Open **Game Scene**, switch to the Game view, and press Play.

## Controls
- Movement: WASD  
- Camera: Mouse look  
- Combat:
  - Left Click / Right Click and Keys 1–3 for abilities

## Player Objective
The player must interact with the AI and trigger all six behaviour states:
Patrol, Chase, Attack, Search, Flee, and Death.

The AI reacts dynamically to the player using a Behaviour Tree system.
